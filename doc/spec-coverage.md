# Spec coverage

import SpecCoverageTable from '@site/src/components/SpecCoverageTable';

Every spec's verification status in one place — the fraction of each spec's success criteria proven by tagged tests, the criteria counts, and how many tests link to it. This is the whole-project view of the same data the bars show inline and each spec page shows in detail. It's regenerated on every docs build, so it reflects current verification, not aspiration.

Sort by any column; click a spec to open its full criteria-by-criteria breakdown. See [How this project works](./ABOUT_THESE_DOCS.md) for what the maturity levels mean.
